# my-gacha-app | 抽卡记录管理系统

一个基于 [Next.js](https://nextjs.org/) 和 [Ant Design](https://ant.design/) 构建的抽卡记录管理与统计分析系统，支持通过 Token 自动拉取或 JSON 导入抽卡数据，提供多维度的统计与可视化。

## ✨ 核心特性

- 🎮 **自动拉取**：输入 Token/Cookies 即可自动获取官方抽卡数据，无需手动整理。
- 📂 **数据导入**：支持上传 JSON 文件或直接粘贴数据，自动去重，不怕重复导入。
- 📊 **首页概览**：总抽数、SSR 数量、UP 命中率、出货间隔等核心指标一目了然。
- 📈 **抽卡统计**：按卡池查看详细出货记录，标记 UP / 歪 / 欧皇附体，运气好不好一看便知。
- 🗂️ **卡池管理**：卡池信息增删改查，支持行内编辑。
- 🔌 **自由适配**：基于接口的适配器模式，可扩展支持更多游戏。
- 🎨 **暗色主题 UI**：深色背景 + 中文界面，护眼舒适。

## 🛠️ 技术栈

- **框架**: [Next.js](https://nextjs.org/)
- **UI 库**: [Ant Design](https://ant.design/)
- **语言**: TypeScript
- **数据库**: [MongoDB](https://www.mongodb.com/)
- **部署**: [Vercel](https://vercel.com/)
